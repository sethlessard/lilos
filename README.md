# lil os

This is currently only a kernel.. working to be an OS.

```bash
sudo apt-get apt-get build-essential gcc make qemu-system
```

### Build the operating system

```sh
./build.sh
```

### Run the operating system in Qemu

```sh
./build.sh

make run
```

## Run the ISO in Qemu

```sh
./build.sh

make runiso
```

### Clean the project

```sh
./clean.sh
```
